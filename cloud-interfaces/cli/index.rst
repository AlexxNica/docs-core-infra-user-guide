CLI: command-line interfaces and tools
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

by typing commands or executing shell scripts,
you can use command-line interfaces (CLIs) 
provided by OpenStack and, 
in some cases, supported by Rackspace.
