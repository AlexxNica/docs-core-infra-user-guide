Cloud Images and the Cloud Control Panel
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

as described at :ref:`cloud_tour`, 
with 
Cloud Servers (labeled *Cloud Servers*) 
and Cloud Images (labeled *Saved Images*) both available 
under *Servers*.
